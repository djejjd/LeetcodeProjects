### 1. Two Sum (LeetCode 1) - Easy
- **难度**: Easy
- **链接**: [LeetCode #1](https://leetcode.com/problems/two-sum/)
- **完成时间**: 2026-04-25 18:00
- **状态**: ✅ 完成
- **文件**: `solutions/easy/001_two_sum.py`

**关键点:**
- 使用哈希表优化暴力解法
- 时间复杂度从 O(n²) 降低到 O(n)
- 空间复杂度 O(n)

**解题思路:**
```python
1. 创建空字典 seen = {}
2. 遍历数组中的每个数字
3. 计算补数 = target - 当前数字
4. 如果补数在 seen 中，返回 [seen[补数], 当前下标]
5. 否则将当前数字加入 seen
```

**测试用例:**
```
输入: nums = [2,7,11,15], target = 9
输出: [0, 1]
✅ 通过

输入: nums = [3,2,4], target = 6
输出: [1, 2]
✅ 通过
```
